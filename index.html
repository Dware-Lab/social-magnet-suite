Friend maker

Friend Maker is a chrome extension that helps you to automate the process of sending friend requests. Using Friend Maker you can send requests to people inside a group, friend list of a person, suggested friends, people reacted on a post and comment. You can also use Friend Maker to send a personalized message (with their first and last name) right after you send a request to more people accepting your request. When you add friends you can set different conditions to make sure you are adding targeted friends like the delay between sending a request, number of mutual friends a person must have to send a request, keywords that should be seen on their profile.
